# To-do-list - api
nodejs, express, jwt, bcrypt, jest, cucumber, mysql

# Requirements
- https://github.com/dariusz-m/to-do-list-web-ui
- Node v8.11.1 +
- npm 6.0.1 +
- docker
- docker-compose

https://nodejs.org

# Development
### Install dependencies
- npm install
### Running application
- cd docker && docker-compose up  // To run mysql database
- npm start
